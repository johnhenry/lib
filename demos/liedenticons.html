Liedenticons

SVG

Basic Usage (SVG)

Specify just a hash, use default background color, padding, and size. Foreground colors is based on input.

Options

Include an options object to customize foreground color, background color, padding, size.

As an Image Source

The "toString" method takes a boolean. When set, it will append a preamble to the string. This is useful when embedding as source for an image.

As an Image Source (base 64)

The "toString()" method takes a second boolean. When set to true, the output is a base64 string rather than svg text.

PNG

Basic Usage (PNG)

The generated string is always in base64 format. By default, the preamble is attached.

Raw String

Pass "false" into to "toString" to drop preamble.
