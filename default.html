Standard Library

This is my standard library.

You can find the source code here at https://github.johnhenry.com/lib

It's a collections of various modules, components, and applications that I've created over the years.

By publishing them here, I hope that you'll use them in your projects too!

/bash contains useful utitlities that I use almost daily.

/css contains useful CSS utilities. Some contain useful class that you can attach to html elements such as the "hide-n-show-landscape" module, which hides and shows elements at the right time. Other modules provide a univresal effect such as "universal-box-sizing" or "universal-unstyled-links". Still, other moduels provide useful variables such as the "colors-tango" library.

All CSS modules are available in plain, uncompliled code and can be imported directly into any modern web application without modification.

/js contains useful JavaScript utilities. Like the CSS modules, thes are coded in plain, uncompliled code and can be imported into directly into any modern web applicion OR used in a deno application (depending upon whether or not the proper globals are available.)

The documentation is non-existent at the moment; but you can see how to use some of the libraies by examing the source code of some of my apps. Check out the code for forsnaken Additionally, I'm working on some demos to demonstrate how they work.
