Hide and show / show and hide

Apply the `hide-and-show` class to hide an element and show it under certain conditions.

Apply the `show-and-hide` class to show an element and hide it under certain conditions.

Apply the `hide-and-show-portrait` to a class with 'hide-and-show' to show it in portrait mode.

Apply the `show-and-hide-landscape` to a class with 'show-and-hide' to hide it in landscape mode

Apply the `hide-and-show-portrait-{stop1}-{stop2}` to a class with 'hide-and-show' to show it between stop1 and stop 2.

  • `-{stop}` works for everyting after to the stop
  • `--{stop}` works for everyting up to the stop

Stops are 0, 256px, 512px, and 1024px pixels.
