# Universal Border Debug

CSS debugging tool color borders red, green, or blue.

This is useful for visual debugging and should not be used in production code.

Usage:

<link rel="stylesheet" href="../universal-border-debug/0.0.0/red.css" /> <!-- OR --> <link rel="stylesheet" href="../universal-border-debug/0.0.0/green.css" /> <!-- OR --> <link rel="stylesheet" href="../universal-border-debug/0.0.0/blue.css" />
